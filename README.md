DJ 턴테이블 시뮬레이터
=====================

노트북으로 턴테이블 돌려보는 가벼운 장난감입니다. MP3나 WAV를 올려서 바로 스크래치해 보세요.

## 바로 돌려보기
```bash
npm install
npm run dev
```
터미널에 뜨는 `http://localhost:5173` 주소로 접속하면 끝.

## 이렇게 놀아요
- 첫 화면에 MP3/WAV 파일을 선택하거나 드래그합니다.
- 왼쪽 위 전원 노브를 켜고 START 버튼으로 재생.
- 바이닐을 드래그하면 스크래치, 반대로 돌리면 역재생.
- 오른쪽 노브/슬라이더로 볼륨·피치를 조절해 미세 튠.

## 빌드/확인
```bash
npm run build
npm run preview   # 번들 확인용
```
